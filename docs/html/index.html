class pymuscle.Muscle(motor_unit_count)[source]¶

A user-created Muscle object.

Used to simulate the input-output relationship between motor neuron excitation and muscle fiber contractile state over time.

Parameters:
  • motor_unit_count – How many motor units comprise this muscle.
  • history_size – The window over which we calculate fatigues

Usage:

>>> from pymuscle import Muscle
>>> muscle = Muscle(60)
>>> input = np.rand(60,1)
>>> muscle.step(input, 1 / 50)
